## 如何更新网站

1. 修改 `index.html` 文件中的文字内容
2. 替换 `images/` 目录下的图片
3. 提交代码：`git add . && git commit -m "更新网站"`
4. 推送到 GitHub：`git push`
5. 等待 1-2 分钟，GitHub Pages 自动部署

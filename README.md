# COCO 个人网站

这是一个基于 GitHub Pages 的个人网站项目。

## 网站地址

部署后的访问地址：`https://LIYIFEI1424.github.io/COCO`

## 项目结构

```
COCO/
├── index.html    # 网站首页
├── README.md     # 项目说明
└── .gitignore    # Git 忽略文件
```

## 如何更新网站

1. 修改 `index.html` 文件
2. 提交代码：`git add . && git commit -m "更新网站"`
3. 推送到 GitHub：`git push`
4. 等待 1-2 分钟，GitHub Pages 自动部署

## 技术栈

- CSS3 (内联样式)
- 原生 JavaScript
